Sistema de Gerenciamento de Produtos 🛒

Este é um sistema em Python desenvolvido para cadastrar, listar e gerar relatórios de produtos nacionais e importados.

🚀 Funcionalidades
- Cadastro de produtos com validação de preço e estoque.
- Suporte a produtos importados (com identificação do país de origem).
- Listagem completa e detalhada dos itens.
- Relatório final com média de preços, item mais caro/barato e valor total do estoque.
